NAICS Classifier (2017)

Paste a business description (up to 10,000 characters). This calls POST /classify and shows the top 3 NAICS codes.

API docs
Request JSON:
{ "text": "your business description" }
Example curl:
curl -X POST /classify \
  -H 'content-type: application/json' \
  -d '{"text":"We install and repair residential and commercial roofs..."}'
Tip: Open DevTools → Network if you want to see the raw JSON response.
